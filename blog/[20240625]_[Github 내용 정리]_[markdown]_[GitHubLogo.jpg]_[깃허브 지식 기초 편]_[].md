# 버전관리 시스템

속 없는 붕어빵이 없는 것 처럼 개발자에게는 버전관리 시스템이란 땔래야 땔 수 없다. 그 중에서도 GitHub(github, Github 아님...)는 붕어빵 중에서도 팥붕어빵이라고 할 수 있다!
